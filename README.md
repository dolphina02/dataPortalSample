# 🚀 Data Portal - Vue 3

라이나생명 데이터 분석 플랫폼을 위한 현대적이고 직관적인 웹 애플리케이션입니다.

![Vue 3](https://img.shields.io/badge/Vue-3.4.0-4FC08D?style=flat-square&logo=vue.js)
![Vite](https://img.shields.io/badge/Vite-5.0.0-646CFF?style=flat-square&logo=vite)
![License](https://img.shields.io/badge/License-MIT-green?style=flat-square)

## ✨ 주요 기능

### 🏠 **홈 대시보드**
- **실시간 업데이트 알림**: 새로운 대시보드, 데이터, 기능 추가 소식
- **빠른 액션**: 새 대시보드 생성, 데이터 가져오기, SQL 쿼리 등
- **플랫폼 현황**: 실시간 통계 및 성능 지표
- **최근 활동**: 사용자 활동 타임라인
- **인기 데이터셋**: 자주 사용되는 데이터 목록

### 🔍 **데이터 탐색**
- **고급 검색**: 다중 필터 및 정렬 옵션
- **그리드/리스트 뷰**: 사용자 선호에 따른 표시 방식
- **실시간 검색**: 즉시 결과 표시
- **태그 기반 필터링**: 카테고리별 데이터 분류

### 📊 **대시보드**
- **실시간 메트릭**: 핵심 성과 지표 모니터링
- **인터랙티브 차트**: 동적 데이터 시각화
- **사용자 활동 추적**: 실시간 사용자 현황
- **시스템 상태**: 서버 및 서비스 모니터링
- **최근 활동 타임라인**: 플랫폼 내 활동 기록

## 🛠️ 분석 도구

### 💻 **SQL Editor**
- **구문 강조**: SQL 쿼리 하이라이팅
- **스키마 브라우저**: 데이터베이스 구조 탐색
- **쿼리 히스토리**: 실행 기록 관리
- **결과 내보내기**: 다양한 형식으로 결과 저장
- **실시간 실행**: 즉시 쿼리 결과 확인

### ⚡ **Text To SQL**
- **자연어 처리**: 한국어를 SQL로 자동 변환
- **AI 모델 선택**: GPT-4, Claude-3, Gemini Pro 지원
- **스키마 컨텍스트**: 데이터베이스 구조 기반 정확한 쿼리 생성
- **신뢰도 표시**: AI 생성 결과의 정확도 표시
- **쿼리 설명**: 생성된 SQL의 실행 단계 설명
- **생성 히스토리**: 이전 변환 결과 관리

### 🎤 **STT 키워드 검색**
- **음성 파일 업로드**: 드래그 앤 드롭 지원
- **다중 포맷 지원**: MP3, WAV, M4A, OGG
- **오디오 플레이어**: 재생, 일시정지, 구간 이동
- **실시간 STT 처리**: 음성을 텍스트로 변환
- **키워드 추출**: AI 기반 핵심 키워드 자동 추출
- **세그먼트 분석**: 시간대별 음성 인식 결과
- **검색 연동**: 추출된 키워드로 데이터 검색

## 🚀 시작하기

### 필수 요구사항
- Node.js 18.0.0 이상
- npm 또는 yarn

### 설치 및 실행

```bash
# 저장소 클론
git clone <repository-url>
cd vue-data-portal

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```

### 개발 서버
개발 서버는 기본적으로 `http://localhost:3000`에서 실행됩니다.

## 📁 프로젝트 구조

```
src/
├── components/           # Vue 컴포넌트
│   ├── HomeView.vue     # 홈 대시보드
│   ├── SearchView.vue   # 데이터 탐색
│   ├── QueryView.vue    # SQL 에디터
│   ├── TextToSqlView.vue # Text-to-SQL 도구
│   ├── SttSearchView.vue # STT 키워드 검색
│   ├── ModelManagementView.vue # 모델 관리
│   ├── DashboardView.vue # 대시보드
│   ├── ApiView.vue      # API 탐색기
│   └── IconSystem.vue   # 아이콘 시스템
├── style.css            # 글로벌 스타일
├── App.vue              # 메인 애플리케이션
└── main.js              # 애플리케이션 진입점
```
